# Delete Link

<HttpMethodBadge method="DELETE" /> Deletes a link.

### Request
#### Query Parameters
| Name    | Type   | Required | Notes                        |
|---------|--------|----------|------------------------------|
| link_id | string | true     | The ID of the link to delete |

#### Example
```shell
curl \
-H 'Authorization: Bearer {API_KEY}' \
-X DELETE \
https://prettyurl.net/api/v1/links/9e968c2c-62d3-4d20-8899-6601721f25c6
```
