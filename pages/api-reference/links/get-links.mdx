# List Links

<HttpMethodBadge method="GET" /> Returns a list of all your links.

### Request
#### Query Parameters
<PaginationQueryParameters />

#### Example
```shell
curl \
-H 'Authorization: Bearer {API_KEY}' \
-X GET \
https://prettyurl.net/api/v1/links
```

### Response
#### Schema
<PaginationResponseSchema />

`links` *object*
| Key          | Type    |
|--------------|---------|
| id | string |
| title  | string |
| domain        | string |
| group     | object |
| back_half     | string |
| short_link     | string |
| original_link     | string |
| updated_at     | datetime |
| created_at     | datetime |
